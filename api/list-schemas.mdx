---
title: List Schemas
description: List all schemas within a share
---

This API returns a paginated list of schemas contained within a specific share. A schema is a logical grouping of tables within a share.

## Request

<ParamField path="method" type="string" required>
  GET
</ParamField>

<ParamField path="url" type="string" required>
  `{prefix}/shares/{share}/schemas`
</ParamField>

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  ```
  Authorization: Bearer {token}
  ```
</ParamField>

### URL Parameters

<ParamField path="share" type="string" required>
  The share name to query. This parameter is case-insensitive.
</ParamField>

### Query Parameters

<ParamField query="maxResults" type="Int32">
  The maximum number of results per page that should be returned.
  
  - Must be non-negative
  - 0 will return no results but `nextPageToken` may be populated
  - Server may return fewer items even if more are available
</ParamField>

<ParamField query="pageToken" type="String">
  Specifies a page token to use. Set to the `nextPageToken` returned by a previous list request to get the next page of results.
</ParamField>

## Response

<ResponseField name="items" type="array">
  Array of schema objects. May be an empty array or missing when no results are found.
  
  <Expandable title="Schema Object">
    <ResponseField name="name" type="string" required>
      Schema name (case-insensitive). Must not exceed 255 characters.
    </ResponseField>
    
    <ResponseField name="share" type="string" required>
      The parent share name. Must not exceed 255 characters.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  Token for retrieving the next page of results. May be an empty string or missing when there are no additional results.
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET \
  '{prefix}/shares/vaccine_share/schemas?maxResults=10&pageToken=...' \
  -H 'Authorization: Bearer {token}'
```

```python Python
import requests

url = "{prefix}/shares/vaccine_share/schemas"
headers = {
    "Authorization": "Bearer {token}"
}
params = {
    "maxResults": 10,
    "pageToken": "..."
}

response = requests.get(url, headers=headers, params=params)
schemas = response.json()
```

```javascript JavaScript
const response = await fetch(
  '{prefix}/shares/vaccine_share/schemas?maxResults=10&pageToken=...',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer {token}'
    }
  }
);

const schemas = await response.json();
```
</CodeGroup>

## Example Response

```json
{
  "items": [
    {
      "name": "acme_vaccine_data",
      "share": "vaccine_share"
    }
  ],
  "nextPageToken": "..."
}
```

<Note>

</Note>
